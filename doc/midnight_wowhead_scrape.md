# PAGE 6: TRANSMOG UI — Everything About Transmogrification Appearances
> Source: https://www.wowhead.com/guide/transmogrification/everything-about-transmog-ui-appearances

## CORE SYSTEM MECHANICS

### Slot-Based Transmogrification
- Transmogrification affects gear **slots** instead of the gear itself
- Appearance changes persist when gear is upgraded/swapped
- Changes apply to armor slots rather than individual items

### Warband Integration
- Obtaining gear on **any** character adds its appearance to library
- Appearance only **usable** by characters wearing that armor type
- Quest completion unlocks **all** appearance rewards regardless of armor type

### Tier Set Unlock Rule (Midnight Pre-Patch)
- Collecting every appearance on a single difficulty for a tier set also unlocks appearances for any easier difficulties

## OUTFIT SLOT COSTS

| Slot # | Gold Cost |
|--------|-----------|
| 1-2 | Free (default) |
| 3 | 100 |
| 4 | 200 |
| 5 | 350 |
| 6 | 500 |
| 7 | 1,000 |
| 8 | 2,000 |
| 9 | 3,500 |
| 10 | 5,000 |
| 11-15 | 10,000 each |
| 16-20 | 20,000 each |
| 21-30 | 35,000 each |
| 31-40 | 50,000 each |
| 41-50 | 100,000 each |
| **Total (50 slots)** | **2,012,650 gold** |

**Update cost**: Up to 31,200,000 gold to update all slots simultaneously (includes weapons + illusions)
**Swap cost**: Free once outfits are saved

## OUTFIT TYPES

### Outfits
- Require gold to create and modify
- Equippable as action bar buttons for no-cost swapping
- Support situation-based automatic changes
- Can be "locked" to prevent situation overrides

### Custom Sets
- Zero gold cost to create
- Do NOT change character appearance directly
- Enable saving outfit ideas without commitment
- Automatically converted from legacy transmog sets
- Applied to outfit slots at transmogrifier for gold fee

## SITUATIONS SYSTEM
Automatic transmog changes based on conditions. ALL conditions must be met for activation. If multiple outfits qualify, one is randomly selected.

### Location Conditions
- Rest Areas
- House
- Character Select
- World
- Delves
- Dungeons
- Raids
- Battlegrounds

### Movement Conditions
- Unmounted
- Mounted
- Swimming
- Ground Mount
- Flying Mount

### Other Conditions
- Specialization
- Equipment Set
- Racial Form (Worgen: Worgen/Human; Dracthyr: Dracthyr/Visage)

**Situation adjustment cost**: Free to adjust conditions or lock outfits

## UI FEATURES

### Collections Tab — Appearances
- Filter by: gear slot, class, faction, race, source, collection status
- Search by item name
- Shift-click missing appearances to create map pins
- View complete sets by class
- Identify missing set pieces

### Transmogrifier NPC Interface
- Unlock and modify outfit slots
- Click gear slots to change appearance
- Shoulder slots: change left/right separately
- View and equip official sets
- View and equip custom sets
- Filter and search appearances by name

### Accessible Via
- Major city Transmogrifier NPCs
- Grand Expedition Yak mount (Item 122708)
- Ethereal Transmogrifier toy (Item 206268)

## SLOT OPTIONS

### Hide Slot
- Hidden Helm
- Hidden Cloak
- Other armor slots supported
- **Limitation**: Weapons CANNOT be hidden

### Other Slot Options
- "Ignore Slot": Prevents equipment slot from displaying
- "Show Equipped Gear": Displays current gear appearance instead of transmog

## WOWHEAD DRESSING ROOM TOOL
- External tool at wowhead.com/dressing-room
- Barbershop customization (race, features, hairstyle, accessories, DK colors)
- Character import via Profiler
- Browse gear by name or image
- Apply complete transmog sets
- Animation support and mount posing
- Automatic URL-based saving
- Permanent custom profile option

- Weapon transmog challenges (1H to staff conversions)
- Situation activation inconsistencies
- Hidden slot functionality gaps
